# 🎿 SMM Agent Bot — Андорра v2.0

Telegram-бот для автоматической генерации постов для 4 брендов горнолыжного бизнеса в Андорре.

## Бренды

| Бренд | Файл промпта | Описание |
|-------|-------------|----------|
| 🏔 Andorra Club | `prompts/club.md` | Туристическое агентство — local expert по Андорре |
| 🎿 Инструктор | `prompts/instructor.md` | Личный бренд инструктора, 30 лет опыта |
| ⛷ Прокат | `prompts/rent.md` | Прокат горнолыжного оборудования |
| 🏫 Школа | `prompts/school.md` | Горнолыжная школа для детей и взрослых |

## Как это работает

```
Вы → бот → выбор бренда → выбор платформ
→ описание темы + тип поста (пост / рилс / история / видео)
→ прикрепляете медиа или выбираете генерацию
→ Claude AI генерирует все посты параллельно
→ предпросмотр каждого поста → одобрить / изменить / перегенерировать / пропустить
→ выбор публикации: сейчас / запланировать / черновик
→ готово!
```

## Поддерживаемые платформы

| Платформа | Фото | Видео | Документ | Типы постов |
|-----------|------|-------|----------|-------------|
| 📸 Instagram | до 3 | 1 Reels | — | пост, рилс, история |
| 👥 Facebook | до 3 | 1 | ✅ | пост, история, видео |
| 🎵 TikTok | 1 (обложка) | 1 | — | видео, слайдшоу |

> тогда бот сохранит оригинальное разрешение без сжатия.

## Расписание публикаций

- 🟢 **Сейчас** — немедленная публикация в Telegram-канал
- 🕐 **Запланировать** — выбор даты и времени (часовой пояс Андорры UTC+2)
- 📋 **Черновик** — сохранить без публикации

Планировщик проверяет запланированные посты каждую минуту.

## Установка

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/alexmonitor471/ai-smm-agent.git
cd ai-smm-agent
```

### 2. Создайте .env файл

```bash
cp .env.example .env
```

Заполните переменные (см. раздел «Переменные окружения» ниже).

### 3. Запуск локально

```bash
pip install -r requirements.txt
python bot.py
```

### 4. Запуск через Docker

```bash
docker build -t smm-bot .
docker run --env-file .env smm-bot
```

### 5. Деплой на Railway (рекомендуется)

1. Зайдите на [railway.app](https://railway.app)
2. **New Project → Deploy from GitHub repo**
3. Выберите репозиторий `ai-smm-agent`
4. Добавьте PostgreSQL: **+ New → Database → PostgreSQL**
5. Добавьте переменные окружения
6. Deploy!

## Переменные окружения

```env
# Telegram Bot (получить у @BotFather)
BOT_TOKEN=ваш_токен

# Ваш Telegram ID (узнать у @userinfobot)
# Несколько через запятую: 123456,789012
ALLOWED_USER_IDS=ваш_id

# Anthropic Claude API (console.anthropic.com)
ANTHROPIC_API_KEY=ваш_ключ

# PostgreSQL (Railway подставит автоматически)
DATABASE_URL=postgresql://user:pass@host:5432/dbname

# Replicate — генерация AI-фото (replicate.com, опционально)

# Telegram-каналы брендов (ID или @username канала)
TG_CHANNEL_CLUB=@ваш_канал_агентства
TG_CHANNEL_INSTRUCTOR=@ваш_канал_инструктора
TG_CHANNEL_RENT=@ваш_канал_проката
TG_CHANNEL_SCHOOL=@ваш_канал_школы
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Начать новый пост |
| `/new` | Новый пост (сброс сессии) |
| `/history` | История последних 10 постов текущего бренда |

## Процесс создания поста — подробно

1. `/new` → выбираете **бренд** (кнопки)
2. Выбираете **платформы** (чекбоксы, можно несколько)
3. Пишете **описание**: тема + тип (пост / рилс / история / видео)
4. Прикрепляете **медиа** или выбираете AI-генерацию
5. Бот генерирует все посты **параллельно** (~5–10 сек)
6. Для каждого поста по очереди: **одобрить / изменить / перегенерировать / пропустить**
7. Выбираете **когда публиковать**: сейчас / запланировать / черновик

### Режимы работы с постом

| Кнопка | Действие |
|--------|----------|
| ✅ Одобрить | Пост принят, переходим к выбору времени публикации |
| ✏️ Изменить | Напишите правки текстом — Claude перепишет |
| 🔄 Перегенерировать | Новый вариант с нуля |
| ⏭ Пропустить | Сохранить как skipped, перейти к следующей платформе |
